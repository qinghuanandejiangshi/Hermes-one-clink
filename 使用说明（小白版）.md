# Hermes AI 助手 — Windows 一键安装使用说明

> 本说明适合完全不懂技术的普通用户，跟着步骤走即可，全程无需手敲任何命令。

---

## 一、你需要准备什么

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10（2004 及以上）或 Windows 11 |
| 内存 | 建议 8 GB 以上 |
| 磁盘空间 | 至少 10 GB 可用空间 |
| 网络 | 首次安装需联网（或使用已下载好的离线包） |
| 权限 | 需要**管理员权限**（程序会自动弹窗申请，点「是」即可） |
| API Key | 至少准备一个，推荐国内用户使用 **DeepSeek**（免费额度足够日常使用） |

### 如何获取 DeepSeek API Key（国内推荐）
1. 打开浏览器访问：https://platform.deepseek.com/api_keys
2. 注册/登录账号
3. 点击「创建 API Key」
4. 复制那串 `sk-` 开头的字符串，保存好备用

## 二、首次安装（只需做一次）

### 第 1 步：双击启动安装程序

找到文件夹中的 **`一键启动.bat`**，**右键 → 以管理员身份运行**。

> 如果弹出「是否允许此程序更改电脑」窗口，点击**「是」**。

程序会打开一个黑色窗口，自动完成以下所有事情（耗时约 10–30 分钟，取决于网速）：

```
✅ 第 1 步：检查系统环境
✅ 第 2 步：安装 WSL2（Windows Linux 子系统）
✅ 第 3 步：安装 Ubuntu
✅ 第 4 步：安装 Hermes AI 核心
✅ 第 5 步：安装 Web 界面
✅ 第 6 步：配置 API 密钥
✅ 第 7 步：启动服务并打开界面
```

### 第 2 步：填写 API Key

安装到第 6 步时，会弹出一个配置窗口：

![API Key 配置窗口示意](docs/screenshot-apikey.png)

- **OpenRouter Key**（可选）：在 openrouter.ai 注册获取，可调用多种模型
- **OpenAI Key**（可选）：在 platform.openai.com 获取
- **DeepSeek Key**（推荐国内用户填写）：填入 `sk-` 开头的字符串

> 至少填一个，其余可以留空。填完点「保存并继续」。

### 第 3 步：等待浏览器自动打开

安装完成后，浏览器会自动打开 `http://127.0.0.1:8787`，进入 Hermes Web 界面。

**首次打开会有一个配置向导**，按提示：
1. 选择模型提供商（填了 DeepSeek Key 就选 DeepSeek）
2. 设置工作目录（默认即可，直接 Continue）
3. 完成后进入主界面

---

## 三、日常使用（后续每次启动）

安装完成后，以后每次使用只需：

**双击 `后续启动.bat`**（同样需要以管理员身份运行）

程序会自动启动后台服务并打开浏览器。

## 四、Web 界面使用简介

打开 `http://127.0.0.1:8787` 后，你会看到：

```
┌─────────────────────────────────────────────────┐
│  左侧边栏                  主聊天区域              │
│  • Sessions（对话列表）                           │
│  • Files（文件管理）                              │
│  • Settings（设置）       输入你的问题，回车发送   │
└─────────────────────────────────────────────────┘
```

### 常用功能

- **新建对话**：点击左上角 「+」按钮
- **选择模型**：顶部下拉菜单切换 DeepSeek / OpenAI 等
- **上传文件**：拖拽文件到聊天框，可分析代码、文档
- **工作区**：点击 Files 可管理 AI 可以操作的本地文件夹
- **切换模型**：在 Settings → Profiles 中添加自定义模型端点（如 Kimi）

### 添加 Kimi（月之暗面）等国内模型

1. 进入 **Settings → Profiles → New Profile**
2. 填写：
   - Base URL: `https://api.moonshot.cn/v1`
   - API Key: 你的 Kimi API Key
   - Model: `moonshot-v1-8k`
3. 保存后即可在顶部下拉选择

## 五、常见问题

### Q：安装卡在某一步很久，怎么办？
A：耐心等待，网络慢时安装 Linux 包可能需要 15 分钟以上。如果超过 30 分钟没有任何进展，关闭窗口重新运行 `一键启动.bat`，程序会从断点继续。

### Q：弹出「执行策略」或「安全警告」怎么办？
A：这是 Windows 的正常安全提示。出现选项时选 **「A（全是）」** 即可继续。

### Q：浏览器打开后显示「Connection lost」？
A：等 5–10 秒，页面会自动重连。如仍无法使用，关闭浏览器，重新双击 `后续启动.bat`。

### Q：忘记 API Key 了，怎么修改？
A：重新运行 `一键启动.bat`，程序检测到已安装后会提示「是否重新配置 API Key」。

### Q：提示「Cross-origin request rejected」？
A：关闭浏览器，改用**无痕/隐私模式**打开 `http://127.0.0.1:8787`（Chrome: `Ctrl+Shift+N`）。

### Q：想彻底卸载怎么办？
A：打开 PowerShell（管理员），执行：
```
wsl --unregister Ubuntu
```
然后在「设置 → 应用」中卸载「适用于 Linux 的 Windows 子系统」即可。

## 六、技术说明（可选阅读）

本工具安装的组件：
- **WSL2**：Windows 内置的 Linux 虚拟层，Hermes 运行在其中
- **Ubuntu**：轻量级 Linux 系统（运行在 WSL2 内，不影响 Windows）
- **Hermes Agent**：AI 代理核心，通过 uv 包管理器安装
- **Hermes WebUI**：本地 Web 界面，运行在 `127.0.0.1:8787`
- **API Key**：存储在 WSL2 Ubuntu 的 `~/.bashrc` 中，不会上传任何服务器
